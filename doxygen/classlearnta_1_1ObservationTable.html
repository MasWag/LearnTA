bool learnta::ObservationTable::exteriorSaturate ( )
inline

Make the observation table exterior-saturated.

Observation table is exterior-saturated if for any \(p \in P\), if \(\mathrm{ext}^t(p) \not\in P\), we have \(\mathrm{ext}^t(p) \sim^{S, \top} p\).

Returns
If the observation table is already exterior-saturated
